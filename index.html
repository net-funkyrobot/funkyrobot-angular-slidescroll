What it does

Angular slidescroll enables a full-screen, slide based layout with keyboard and scroll slide transition.

funkyrobot-angular-slidecroll is an Angular JS directive that can be installed as a JavaScript component with Bower.

Slide transitions are driven by JavaScript events which makes hooking up custom slide controls easy.

Using the directive is easy! Just create a <slidescroll></slidescroll> element within an Angular JS app (inside the element with the ng-app="myApp" attribute).

Within <slidescroll></slidescroll>, create <slide></slide> elements and place your slide contents within this.

Finally, within your Angular JS app's module definition, specify frSlidescroll as a dependency.

Angular slidescroll supports basic transition controls

To scroll up press the up key, page up key or scroll upwards.

To scroll down press the down key, page down key, press space or just scroll downwards.

The home and end keys move to the first and last slides respectively.

I need an example

Angular JS module definition

angular.module('demoApp', ['frSlidescroll']);
<head>
    <!-- Angular slidescroll required resources -->
    <script type="text/javascript" src="angular.js"></script>
    <script type="text/javascript" src="angular-slidescroll.js"></script>
    <link rel="stylesheet" type="text/css" href="angular-slidescroll.css" />

    <!-- Demo related resources -->
    <script type="text/javascript" src="demo.js"></script>
    <link rel="stylesheet" type="text/css" href="demo.css">
</head>
<body ng-app="demoApp">
    <slidescroll>
        <slide>
            <h2>Slide #1</h2>
        </slide>
        <slide>
            <h2>Slide #2</h2>
        </slide>
    </slidescroll>
</body>

Overlays, headers and footers

You can display elements as overlays, on-top of slides.

Ensure any overlay element is defined after the <slidescroll> element. Make sure it is positioned as a fixed element and given a position relative to the top, bottom, left or right of the browser window. 

<slidescroll>
    ...
</slidescroll>

<div id="header">
    <div class="container">
        <h1>Funkyrobot Angular Slidescroll UI</h1>
        <h4>An Angular JS directive</h4>
    </div>
</div>

#header {
    position: fixed;
    top: 0;
    height: 120px;
    width: 100%;
}

Slide transition event interface

The buttons on the panel below are custom transition controls which use the event driven interface.

Each button's click handler simply broadcasts either a TransitionEvent or TransitionToEvent event using Angular's $scope.$broadcast() function. 

// 'TransitionEvent', slideDelta (difference from current slide)
$scope.$broadcast('TransitionEvent', -1);

// 'TransitionToEvent', slideIndex (index of the slide to transition to)
$scope.$broadcast('TransitionToEvent', 2);

If the event is fired from outside the directive's scope, from your app's controller for example, use Angular JS' $scope.$broadcast() function. If the event is being fired from within the slidescroll element's scope use $scope.$emit().
